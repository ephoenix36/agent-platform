## üöÄ **DEPLOYMENT INSTRUCTIONS**

### **Quick Start (Development)**
```bash
# Terminal 1: Backend
cd apps/api
pip install -r requirements.txt
python -m uvicorn main:app --host 127.0.0.1 --port 8000 --reload

# Terminal 2: Frontend  
cd apps/web
npm install
npm run dev

# Access: http://localhost:3000
# Login: admin@platform.com / admin123
```

### **Production Deployment**
```bash
# 1. Build Frontend
cd apps/web
npm run build
npm start  # Runs on port 3000

# 2. Run Backend
cd apps/api
uvicorn main:app --host 0.0.0.0 --port 8000 --workers 4

# 3. Configure Reverse Proxy (Nginx)
# 4. Set up SSL certificates (Let's Encrypt)
# 5. Configure environment variables
# 6. Set up monitoring (optional)
```

### **Docker Deployment** (Recommended)
```bash
# Build and run both services
docker-compose up -d

# Or manually:
docker build -t agent-platform-api ./apps/api
docker build -t agent-platform-web ./apps/web

docker run -d -p 8000:8000 agent-platform-api
docker run -d -p 3000:3000 agent-platform-web
```
