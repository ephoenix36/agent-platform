# ADR 6: Wave 2 Security Engagement

**Status:** proposed  
**Date:** 2025-11-16

## Context

Parallel tasks include auth and deployment touches; need security oversight.

## Decision

Security agent to run alongside backend/auth agents in Phase 2 to audit input validation, Whop auth, env handling.

## Consequences

Ensures best practices while parallelizing; may flag issues requiring rework.
