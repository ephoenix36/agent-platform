# ADR 54: Wave 2 Agent Orchestration Blocker

**Status:** proposed  
**Date:** 2025-11-16

## Context

Attempted to run parallel agent team for tasks 2-4 but failed due to missing ANTHROPIC_API_KEY.

## Decision

Document blocker and either configure API key or fallback to manual execution.

## Consequences

Parallel automation paused until API key configured.
